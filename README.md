stacklr_tui [![Build Status](https://travis-ci.org/mamewotoko/stacklr_tui.svg?branch=master)](https://travis-ci.org/mamewotoko/stacklr_tui)
==========

Text user interface of stacklr for desktop

Build
-----
```
make
```

Run
---
```
./stacklr_tui
```
press F1 or q key to exit.
